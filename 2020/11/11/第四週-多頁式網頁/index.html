Layout 觀念設計

  • ID 與錨點設計

    • id 只能唯一性
    • 重複id在舊瀏覽器會出錯

  • 什麼是 Layout ？

    • 多頁式共同的地方

SCSS

  • 有順序性！

  • import 檔案切分

    • 最後只會有一隻最終的css

  • 階層

    • ３層是為閱讀性佳的程式碼，而最後一層（第四層）通常保留給擬態選擇器
    • 要管理就要了解編譯
    • YT - 六角：CSS 選擇器優化影片
    • css 可以直接貼到 scss
    • 大學體驗營：常見錯誤

  • 變數

    • 顏色

    • 字型
      • font-s \ font-m \ font-xl

  • import 拆分規劃

    • _：下底線命名開頭，是不做編譯，純用來合併
